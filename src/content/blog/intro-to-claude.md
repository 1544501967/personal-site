# Claude 简介

## 什么是 Claude？

Claude 是由 **Anthropic** 开发的 AI 助手，以安全、诚实和有益为设计理念。它能够理解自然语言、编写代码、分析数据、翻译文本，并协助完成各种复杂任务。

## 核心能力

| 能力 | 说明 |
|------|------|
| **对话理解** | 深度理解上下文，支持多轮对话 |
| **代码生成** | 支持主流编程语言，可编写、调试和重构代码 |
| **文本分析** | 长文档摘要、信息提取、情感分析 |
| **多语言** | 流利支持中、英、日等多种语言 |
| **工具调用** | 通过 MCP 协议与外部工具和服务交互 |
| **视觉理解** | 识别和分析图片中的内容 |

## 模型系列

- **Opus** — 最强推理能力，适合复杂架构设计和深度分析
- **Sonnet** — 最佳编程模型，平衡能力与效率
- **Haiku** — 轻量快速，适合高频调用场景

## 如何使用

Claude 提供多种接入方式：

1. **claude.ai** — 网页端直接对话
2. **Claude Code** — 命令行工具，与代码库深度集成
3. **API** — 通过 Anthropic API 集成到自有应用
4. **IDE 插件** — VS Code、JetBrains 等编辑器内使用

## 设计原则

> AI 应该是**安全的**、**诚实的**和**有帮助的**。

Anthropic 采用"Constitutional AI"方法训练 Claude，使其在保持强大能力的同时，遵循人类价值观和安全准则。
