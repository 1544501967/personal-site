# 个人主页

![首页预览](/projects/personal-site.svg)

一个参考 exa.ai 设计风格的极简个人主页，用于展示项目作品、技术随笔和个人简历。

## ✨ 功能特性

- **首页** — Hero 展示 + 最新随笔 + 精选项目
- **随笔系统** — Markdown 写作，标签分类，代码高亮
- **项目展示** — 卡片式布局，支持图片、GitHub 和演示链接
- **关于页面** — 技能、工作经历、教育背景
- **响应式设计** — 完美适配手机、平板、桌面
- **极简风格** — 大量留白，专业干净，阅读体验优秀

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| Astro | 静态网站生成 |
| Tailwind CSS | 样式系统 |
| MDX | Markdown 增强 |
| Cloudflare Pages | 部署平台 |

## 📐 设计规范

```
主色调: #1E40ED (Exa 蓝)
背景色: #FFFFFF / #FAF9F8
字体:   Inter + JetBrains Mono
风格:   极简 / 专业 / 技术感
```
