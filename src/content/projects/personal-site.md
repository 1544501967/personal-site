# 个人主页

一个简洁、现代的个人主页网站，用于展示项目作品和技术随笔。

## 特性

- 响应式设计，适配各种设备
- Markdown 内容管理
- 深色模式支持
- 极致的加载性能

## 技术栈

- Astro
- Tailwind CSS
- MDX
